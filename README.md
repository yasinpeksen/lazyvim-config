# 💤 LazyVim

My personal template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

### Linux / MacOS
```bash

git clone https://github.com/yasinpeksen/lazyvim-config.git ~/.config/nvim

```


### Windows
```bash

git clone https://github.com/yasinpeksen/lazyvim-config.git $env:LOCALAPPDATA\nvim

```
